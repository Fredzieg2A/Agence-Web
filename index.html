Accueil

Bienvenue sur le site de notre agence web. Nous sommes spécialisés dans la création de sites internet et le référencement. Nous vous accompagnons dans la réalisation de votre projet web.

  • 1. Créer un dossier "Agence Web"
  • 2. Créer un index.html avec du contenu
  • 3. Initialiser le dépôt local et faire un commit
  • 4. Créer un un dépôt distant sut GitHub
  • 5. Lier le dépôt distant et le dépôt local
  • 6. Envoyer le code du dépôt local sur le dépôt distant
  • 7. Créer une branche "Services"
  • 8. Ajouter une page services.html
  • 9. Faire un commit de ces changements
  • 10. Envoyer cette branche sur GitHub
  • 11. Intégrer les changements de cette branche à la branche principale à l'aide d'une pull request
